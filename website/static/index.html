How It Works

The detection pipeline powered by deep learning

01

Upload Image

Upload a parking lot aerial/camera image and select the matching camera profile (1–9).

02

YOLOv5 Detection

The image is processed through YOLOv5s neural network to detect all vehicles in the frame.

03

Slot Matching

Each predefined parking slot is compared against detections using overlap and area thresholds.

04

Results
