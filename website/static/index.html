How It Works

A more automatic pipeline for parking-lot image uploads.

01

Upload Or Paste

Users can browse, drag-and-drop, or paste a parking image without adjusting controls first.

02

Decode & Normalize

The backend handles modern image formats, fixes orientation metadata, and prepares the image for inference.

03

Vehicle Detection

YOLOv5 focuses on vehicle classes only and removes duplicate boxes for cleaner occupancy decisions.

04

Auto Layout Match

The app first tries the known parking layouts, then estimates parking rows and empty slots for general parking-lot images.
